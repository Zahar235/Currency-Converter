# Currency Converter

**Автор:**Захар Шаповал


**Описание:**
Приложение для конвертации валют с графическим интерфейсом. Использует внешний API для получения актуальных курсов валют. Позволяет сохранять историю последних операций.

**Как получить API-ключ:**
1. Перейдите на сайт [exchangerate-api.com](https://www.exchangerate-api.com/).
2. Зарегистрируйтесь и получите бесплатный ключ разработчика.
3. Скопируйте ключ.
4. Откройте файл `currency_converter.py`.
5. Замените строку `self.api_key = 'YOUR_API_KEY'` на ваш реальный ключ в кавычках.
6. Сохраните файл.

**Как использовать:**
1. Запустите `currency_converter.py`.
2. Выберите валюту «Из» и «В» из выпадающих списков.
3. Введите сумму для конвертации в поле ввода.
4. Нажмите кнопку «Конвертировать».
5. Результат появится под кнопкой.
6. Операция будет добавлена в таблицу истории.

**Примеры использования:**
- Конвертация 100 долларов в евро: выберите USD -> EUR и введите 100.
- Проверка на ошибку: попробуйте ввести отрицательное число или текст — программа покажет сообщение об ошибке.
